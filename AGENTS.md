本项目的目标是实现参考 clash-party 实现一个纯命令行代理程序。

Clash Party 的源代码在： `~/code/clash-party-cli`

技术栈：和 clash party 尽可能对齐，使用 typescript 作为主要开发语言

## 提交纪律

完成任何特性修改、bug 修复、文档规则调整或项目配置变更后，必须先运行与改动范围匹配的验证命令；验证通过后，必须在本轮内创建一次符合全局 Conventional Commits 规则的 git commit。

禁止在验证通过且工作区仍有应提交改动时只汇报“未提交”。只有用户明确要求不提交，或改动包含不安全/混杂内容需要人工处理时，才可以停止提交并说明原因。
